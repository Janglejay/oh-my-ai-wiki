# oh-my-little-claude

> Rust 实现的 claw CLI agent harness。

## 简介

oh-my-little-claude 是 UltraWorkers/claw-code 的 Rust 实现版本，提供更快的启动速度和更好的性能。

## 核心功能

- Rust 重实现
- 支持 MiniMax 等多模型提供商
- OpenAI 兼容接口
