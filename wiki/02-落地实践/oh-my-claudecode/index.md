# oh-my-claudecode

> Claude Code 的多 Agent 编排扩展层。

## 简介

oh-my-claudecode 是对 Anthropic Claude Code 的多 Agent 编排扩展，通过 19 个专业化 Agent 实现复杂的软件开发工作流。

## 核心功能

- **Team Pipeline** — team-plan → team-prd → team-exec → team-verify → team-fix
- **Parallel Mode** — ultrawork 模式，多 Agent 并行执行
- **19 个专业化角色** — Explorer、Planner、Architect、Executor 等
