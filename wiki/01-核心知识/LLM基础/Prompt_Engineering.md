# Prompt Engineering

Prompt 工程是设计和优化输入提示词以获得更好输出的实践。

## 核心原则

- **清晰性** — 明确指定任务和期望格式
- **上下文** — 提供足够的背景信息
- **约束** — 设定输出边界和限制条件
- **迭代** — 根据结果持续优化

## 常见模式

- Few-shot learning — 提供示例
- Chain-of-thought — 引导推理步骤
- Role prompting — 指定角色身份
- Structured output — 要求特定格式输出
